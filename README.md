# Portfolio de Aimar Sarachaga

Portfolio personal construido con Jekyll y el tema Chirpy.

## Sobre mí

Técnico Superior en Telecomunicaciones y estudiante de Inteligencia Artificial y Big Data con interés en DevOps/MLOps.

## Desarrollo local

```bash
bundle install
bundle exec jekyll serve --livereload
```

Abre http://127.0.0.1:4000 en tu navegador.
